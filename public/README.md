# Infinitude Frontend

No build step required. Edit files directly and reload.

- `index.html` — main page with all views
- `scripts/app.js` — Alpine.js application logic
- `styles/main.css` — custom styles
- `fonts/` — icon fonts (Glyphicons, FontAwesome)

Dependencies are loaded via CDN (Alpine.js, canvas-gauges, Chart.js, Bootstrap CSS).
